xiaok 3D icon studies

Three static-exportable directions built from the original dotted K geometry. Each scene uses Three.js spheres, volumetric glow sprites, particles, and a rendered background so the exported image keeps the dream-light look.

A / formal brand

Aurora Glass

Cool glass pearls, cyan bloom, and a warm rim light. This is the cleanest option for a primary app icon or release artwork.

B / dream-light

Dream Orbit

Violet-blue nebula light, orbiting dust, and crystal beads. This is the most fantasy-forward direction for an AI assistant identity.

C / CLI identity

Terminal Nebula

Green terminal glow, subtle grid energy, and white-hot centers. This keeps the strongest technical link to xiaok as a CLI.

Exports are square 1024px images. If the page is opened directly from disk, the Three.js module is loaded from jsDelivr; keep an internet connection for the preview/export page.
