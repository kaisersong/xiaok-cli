# xiaok 3D Icon Studies

Source icon: `xiaokicon-source.png`.

Open `xiaok-3d-icons.html` in a browser to preview three Three.js-based icon directions:

- `Aurora Glass`: clean cyan glass beads with warm rim light.
- `Dream Orbit`: violet-blue nebula, orbiting dust, and the strongest dream-light mood.
- `Terminal Nebula`: green terminal glow and subtle technical grid.

Each card has `Export PNG` and `Export WebP` buttons. Exports are 1024px square images.

The page loads Three.js from jsDelivr, so the browser needs internet access when opening the preview/export page.
